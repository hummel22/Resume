ARM INFO

Lead designer for a 5 DOF robotic arm. This arm is used to pick up rocks of varying size as quickly as possible. Multiple methods where created for controlling the Arm

Design Methods

Each angle can be controlled separately
XYZ coordinates can be inputed by a the user and are transferred to corresponding angles - this done using the quaternions and tensor transformations
The attitude of the claw can be set (Ex: always facing down)

ROVER GUI

Part of controlling the robot involved building the Operator Control GUI. This program will provide updates and information for the controllers. The program uses a xbox controller as a joystick and shows visual graphics of the rover, including wheel direction, video direction, and 3D graphics using openGL to represent the arm. The GUI will also show video streams from the rover and allows the operator to switch views between several cameras. This was programmed using C++ and the QT Developer Library.

The Source code and documentation can be found on github here [Link]
