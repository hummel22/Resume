ARM INFO

Lead designer of the control system for a 5 degree of freedom robotic arm. This arm is used to pick up rocks of varying size as quickly as possible. Multiple methods where created for controlling the arm

Design Methods

Foward Kinematics
Each angle is controlled separately
Easist to design, hardest to control

Inverse Kinematics
Arm is placed in a X-Y plane that rotates with the base
Operator can rotate the arm and only has to select the claw location
All joint angles are calculated from the end point

Claw Attitude Lock
The attitude (pointing diraction) of the claw can be locked
Ex: always facing perpindicular to the ground

ROVER GUI

Part of controlling the robot involved building the Operator Control GUI. This program will provide updates and information for the controllers. The program uses a xbox controller as a joystick and shows visual graphics of the rover, including wheel direction, video direction, and 3D graphics using openGL to represent the arm. The GUI will also show video streams from the rover and allows the operator to switch views between several cameras. This was programmed using C++ and the QT Developer Library.

The Source code and documentation can be found on github here [Link]

MATLAB ORBITAL PROJECT

For a personal project and because it involved some of favorite subjects, I created a MATLAB program that can calculate orbits and simulates satellite pointing. The program also can create small random disturbances and oscillations to test different control systems. I made this program to see if I could combime my knowledge from three different classes: Astrodynamics, Spacecraft Attitude and Control, and CAD Control Systems (Control Theory Class).

Description:

MATLAB Program that can be used to calculate orbits, launchs, and transfers
Can display multiple orbits and calculate orbit transfer data, including time to transfer and ground coverage
Shows vector continously pointing to set postiion as it orbits
Artificail oscillations can be added to test control systems
